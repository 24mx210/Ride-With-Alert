## Packages
leaflet | Maps
react-leaflet | React components for Leaflet maps
socket.io-client | Real-time WebSocket communication
lucide-react | Icons (already in base, but emphasizing usage)
react-webcam | For camera preview and recording mock
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes efficiently

## Notes
- Needs Leaflet CSS in index.html or imported in App
- WebSockets connect to the same host
- Video recording will be simulated or use MediaRecorder API if available in browser
- Maps require OpenStreetMap tiles (free)
